# 📱Google Play Store Data Analysis

## 📌Project Overview
This project focuses on analyzing the Google Play Store dataset to discover insights about mobile applications, user ratings, installs, pricing, categories, and app popularity.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Finding Trends & Insights

This project was completed as part of my Data Science Internship at DecodeLabs.

# 📊 Analysis Performed
### ✅Data Cleaning
- Removed null values
- Fixed incorrect data types
- Cleaned Installs, Price, and Size columns
- Removed duplicates
- Handled missing values
### ✅Exploratory Data Analysis
- Most popular app categories
- Highest rated apps
- Free vs Paid apps comparison
- Installs distribution
- Rating distribution
- Category-wise analysis
### ✅Data Visualization
- Count Plots
- Histograms

## 🔍Key Insights
- Most apps on Play Store are free.
- Family and Game categories contain the highest number of apps.
- Apps with higher ratings generally receive more installs.
- Paid apps are fewer compared to free apps.
- Some categories have significantly higher average ratings.
